Provides parsers and interfaces for extracting data from NMEA 0831 sentences.

Parser implementations and interpretation of sentences is based on the information gathered from the following resources:

  1. NMEA sentence information by Glenn Baddeley (2010-03-06)
  2. NMEA data by Dale DePriest (2010-03-06)
  3. NMEA Revealed v2.1 by Eric S. Raymond (2010-03-06)
  4. Wikipedia (2009-02-17)

The resource disclaimers and warnings concerning the information accuracy apply Java Marine API as well.
