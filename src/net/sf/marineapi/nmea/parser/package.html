Contains base classes and parser implementations for supported NMEA sentences.

Notice

Java Marine API is not official NMEA software or related to National Marine Electronics Association. The interpretation of NMEA sentences for this library is based entirely on freely available resources in the internet. Thus, it is NOT guaranteed that the library would conform and follow the standard accurately.
